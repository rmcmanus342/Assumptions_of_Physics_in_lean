# Assumptions_of_Physics_in_lean
## Intro
Hello I decided it was a fun idea to make a translation of the physical mathematics section of the [Assumptions of Physics Book v3.0](https://assumptionsofphysics.org/book/AssumptionsOfPhysicsV3.0.pdf#part.2). I has turned out to be more work than expected and so I decided to make a repo for it so I can point people to this whenever I have questions.

All credit is due to Gabriele Carcassi, and everyone helping him at [Assumptions of Physics](https://assumptionsofphysics.org/). I only make very small changes to make lean more happy.
